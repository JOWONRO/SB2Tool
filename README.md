# SB2Tool
식붕이툴 전체 코드 및 파일

- backuptest 압축파일의 경우, pyinstaller로 식붕이툴을 실행파일로 만들었을 때 생기는 파일들 중, 프로그램 실행에 영향을 끼치지 않아 제외해도 되는 파일들을 모아놓음.
- 혹시 모르니 backup에 있는 것만 삭제하고 릴리즈할 예정.
